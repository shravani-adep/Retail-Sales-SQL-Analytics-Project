# Retail Sales SQL Analytics Project

## Project Overview

This project demonstrates end-to-end SQL development using a Retail Sales dataset. The project covers database design, data cleaning, SQL querying, advanced analytics, stored procedures, triggers, user-defined functions, transactions, indexing, and error handling.

## Objectives

* Design a normalized relational database
* Load and clean raw sales data
* Perform business analysis using SQL
* Demonstrate advanced SQL concepts 
* Build a portfolio-ready SQL Project

## Database Schema

### Customers

* CustomerID (PK)
* Gender
* Age

### Products

* ProductID (PK)
* ProductCategory
* PricePerUnit

### Sales

* TransactionID (PK)
* SalesDate
* CustomerID (FK)
* ProductID (FK)
* Quantity
* TotalAmount
